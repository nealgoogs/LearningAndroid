README - Simple Math Game

Author: Niall Geoghegan

Description
This is a simple math game for Android where the player selects the correct mathematical operator to complete a given equation. The game tracks the player's score and provides feedback on correct and incorrect answers. If the player answers three questions correctly in a row, they receive a randomized praise message.

References/Resources
Android Documentation (developer.android.com)
Kotlin Standard Library

Instructions to Run the App
1.Clone or download the project to your local machine.
2.Open the project in Android Studio.
3.Connect an Android device or start an emulator.
4.Click "Run" to install and launch the app.
5.Select the correct operator for the given math puzzle.
6.View feedback and continue playing to improve your score.
7.Press "Reset" to restart the game.

Enjoy the game!
